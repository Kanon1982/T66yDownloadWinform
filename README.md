本脚本由 c# 和 selenium 编写，并使用Chrome的107版本浏览器下载，别的浏览器不行

草榴社区：t66y.com

本软件根据 `视频的区域` 、`种子下载总量` 、 `论坛帖子的前多少页数` 、 `是否下载破坏版` 批量下载bt种子

为了防止 草榴 发现是爬虫软件，所以爬取速度设置的很慢
如果想快速爬取，请修改 `T66ySettings.xml` 中 `<implicitly_time>` 和 `<sleep_time>`的值。
修改完之后，需重启软件。
